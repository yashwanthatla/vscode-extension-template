# CodeOwl AI Extension

An AI-powered coding assistant for Visual Studio Code that helps developers write better code faster.

## Features

- **AI-Powered Sidebar**: A dedicated sidebar that welcomes you to CodeOwl AI
- **Beautiful Interface**: Modern, responsive design that matches VS Code's theme
- **Easy Access**: Click the robot icon in the activity bar to open the CodeOwl AI panel

## Getting Started

1. Install the extension
2. Look for the robot (ðŸ¤–) icon in the VS Code activity bar (left sidebar)
3. Click on it to open the CodeOwl AI panel
4. Welcome to your AI coding companion!

## How to Use

- Click the robot icon in the activity bar to open the CodeOwl AI sidebar
- The sidebar will display a welcome message and available features
- Use the "Start Coding with AI" button to begin your AI-assisted coding session

## Requirements

- VS Code 1.78.0 or higher
